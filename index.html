WatchList

This tool adds entities to your Contacts, setting standing to neutral then batch applies a new label and standing to all Neutral entities.

Requires .NET Framework 4.5.1

Screenshots


WARNING: To use this tool effectively you must not have any characters/corps/alliances set to Neutral; If you already have entities set to neutral they will be modified.
